# Cyber Phone 🚀
![Tool Preview](Screenshot_cyber_phone.jpg)

A powerful Python tool for tracking and gathering information from phone numbers.

## 📌 Features

– Track phone number location (Country/Region).
– Indentify carrier and network provider.
– Get timezones and local time information.
– Fully compatible with Kali Linux and Ubuntu and Termux.

## 🛠 Installetion & Setup

Before running the tool, use the setup script to install all required packages:

```bash
git clone https://github.com/Lion-EthicaL/CYBER_PHONE.git

cd CYBER_PHONE

chmod +x phone.py

chmod +x setup.sh

./setup.sh
```

## 🚀How to Use

One the installation is complete, you can run the tool using:
 
```bash
python3 phone.py
```
